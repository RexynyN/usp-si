Para rodar, é necessário o Runtime do Dotnet 6.0, que pode ser baixado no link:

https://dotnet.microsoft.com/en-us/download

Para jogar:

> Ir para o diretório "xadrezserver/executavel" e execute "xadrezserver.exe" para rodar o servidor

> Para cada jogar, ir para o diretório "xadrezclient/executavel" e rode "xadrezclient.exe"

> No cliente, clicar em "Criar partida " e acima do tabuleiro, vai estar escrito "Partida: <id>"

> Com esse Id em mãos, no segundo client, use-o na opção "entrar em partida"

> Para jogar, manipule as coordenadas das peças.
